使用文档

1. 基本用法

将原有的企业微信 Webhook 地址进行简单转换即可使用：

类型 地址格式
原地址 https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxx
代理地址 https://md2t.misrivers.cn/https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxx
2. 发送消息示例

使用方式与企业微信 Webhook 完全一致：

curl -X POST \
  'https://md2t.misrivers.cn/https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxx' \
  -H 'Content-Type: application/json' \
  -d '{
    "msgtype": "text",
    "text": {
        "content": "# 系统告警\\n\\n**服务**: API Gateway\\n**状态**: ❌ 异常\\n**时间**: 2024-01-15 10:30:00"
    }
}'
3. 转换逻辑
  • 检测到 Markdown 格式 → 自动转换为 Text 消息
  • 内容过长时 → 转换为 Text 消息附带链接
  • 非 Markdown 格式 → 原文转发
4. Markdown 支持特性
  • 标题 (H1-H6)
  • 粗体 / 斜体
  • 行内代码 / 代码块
  • 列表 (有序/无序)
  • 表格
  • 引用块
5. 数据安全
  • 所有请求数据加密存储
  • Markdown 查看链接有效期 7天
  • 后台管理需要身份验证
